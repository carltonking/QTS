# QTS - Quant Trading Skills

A unified platform for students and professionals to sharpen their quantitative and trading skills in one place.

## Features

- **LeetCode Practice** - Solve coding problems directly on the platform
- **Chess** - Puzzles and play against AI opponents (adjustable ELO levels)
- **Poker** - Learn fundamentals, pattern recognition, and solve poker puzzles
- **Mental Math** - Customize difficulty (time limits, number ranges, operations) and improve calculation speed and accuracy
- **Quant Interviews** - Practice interview-style technical questions
